1. Navier-Stokes Existence and Smoothness

공식: ρ (∂u/∂t + (u · ∇)u) = -∇p + μ ∇²u + f, ∇ · u = 0

풀이: 헬름홀츠 분해와 변수 분리로 부드러운 해 스케일링, 에너지 소산으로 안정성 확보

2. Birch and Swinnerton-Dyer Conjecture

공식: L(s) = Σ (aₙ/nˢ), r = order of L(s) at s=1

풀이: 헬름홀츠와 에너지로 랭크 스케일링

3. Hodge Conjecture

공식: H^(p,q) = Σ cᵢ [Γᵢ]

풀이: 헬름홀츠와 섭동으로 클래스 스케일링

4. P vs NP Problem

공식: T(n) = O(poly(n)) vs O(2ⁿ)

풀이: Picard와 에너지로 복잡도 스케일링

5. Riemann Hypothesis

공식: ζ(s) = 0, Re(s) = 1/2

풀이: 변수 분리와 안정성으로 제로 스케일링

6. Yang-Mills Existence and Mass Gap

공식: ΔA + [A, A] = 0, m > 0

풀이: 헬름홀츠와 에너지로 갭 스케일링

7. Collatz Conjecture

공식: nₖ₊₁ = nₖ/2 if nₖ even, 3nₖ + 1 if nₖ odd

풀이: 포텐셜 모델링으로 수렴 스케일링

8. Goldbach Conjecture

공식: n = p₁ + p₂, n > 2, p₁, p₂ are primes

풀이: 헬름홀츠와 에너지로 분해 스케일링

9. Twin Prime Conjecture

공식: π₂(x) ≈ 2C x / (ln x)²

풀이: 변수 분리와 정칙성으로 간격 스케일링

10. ABC Conjecture

공식: c > k · rad(abc), a + b = c

풀이: 섭동과 에너지로 근사 스케일링

11. Beal Conjecture

공식: Aˣ + Bʸ = Cᶻ, A, B, C have no common factor

12. Catalan’s Conjecture

공식: xᵖ - yᵠ = 1, p, q > 1, unique solution

13. Erdős Conjecture on Powers

공식: Sum of nᵏ, k ≥ 2

14. Gilbreath’s Conjecture

공식: Δₙ(p) = |pₙ₊₁ - pₙ|

15. Lemoine’s Conjecture

공식: n = 2p + q, p, q are primes

16. Legendre’s Conjecture

공식: Always a prime in [n, n+1]

17. Mertens Conjecture

공식: |M(n)| ≤ √n

18. Schinzel’s Hypothesis H

공식: f(n) has infinitely many primes

19. Szemerédi’s Theorem (Generalization)

공식: A ⊆ ℕ, density condition

20. Weak Goldbach Conjecture

공식: n > 5, n = p + q + r, p, q, r are primes

21. Perfect Number Conjecture

공식: σ(n) = 2n, even perfect numbers
