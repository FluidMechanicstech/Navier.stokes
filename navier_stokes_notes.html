📐 나비에-스토크스 방정식 분석

3차원 해 구하기 및 각 항 분리 분석

1 나비에-스토크스 방정식

비압축성 유체에 대한 나비에-스토크스 방정식:

ρ(∂u/∂t + (u·∇)u) = -∇p + μ∇²u + f
변수 정의:
  • • u = (u, v, w) : 속도 벡터 (3차원)
  • • ρ : 밀도 (상수, 비압축성 가정)
  • • p : 압력
  • • μ : 동점성 계수
  • • f : 외력 (예: 중력)
  • • ∇·u = 0 : 비압축성 조건

방정식의 각 항:

관성항
ρ ∂u/∂t
대류항
ρ(u·∇)u
압력항
-∇p
점성항
μ∇²u
외력항
f

2 점성항과 오일러 방정식

점성항이 없는 오일러 방정식:

ρ(∂u/∂t + (u·∇)u) = -∇p + f

점성항 독립 추출:

μ∇²u

3차원에서 각 성분:

∇²u = (∇²u, ∇²v, ∇²w)

∇² = ∂²/∂x² + ∂²/∂y² + ∂²/∂z²

3 나머지 항 추출

3차원 x-방향 성분:

ρ(∂u/∂t + u∂u/∂x + v∂u/∂y + w∂u/∂z) =
-∂p/∂x + μ(∂²u/∂x² + ∂²u/∂y² + ∂²u/∂z²) + fₓ

관성항 제외:

ρ(u·∇)u = -∇p + μ∇²u + f

대류항 제외:

ρ ∂u/∂t = -∇p + μ∇²u + f

압력항 제외:

ρ(∂u/∂t + (u·∇)u) = μ∇²u + f

외력항 제외:

ρ(∂u/∂t + (u·∇)u) = -∇p + μ∇²u
⚠️ 중요: 각 방정식은 비압축성 조건 ∇·u = 0을 만족해야 함

4 편미분 방정식과 연립방정식

점성항의 편미분 (푸아송 방정식):

∇²u = g

3차원 성분별 표현:

∂²u/∂x² + ∂²u/∂y² + ∂²u/∂z² = gₓ

∂²v/∂x² + ∂²v/∂y² + ∂²v/∂z² = gᵧ

∂²w/∂x² + ∂²w/∂y² + ∂²w/∂z² = g_z
비압축성 조건:
∇·u = ∂u/∂x + ∂v/∂y + ∂w/∂z = 0

5 3차원 해 구하기

가정 (스토크스 흐름):
  • ✓ 정상 상태: ∂u/∂t = 0
  • ✓ 외력 없음: f = 0
  • ✓ 낮은 레이놀즈 수 (대류항 무시)

단순화된 방정식:

-∇p + μ∇²u = 0, ∇·u = 0

압력에 대한 푸아송 방정식 유도 (발산 취함):

∇·(-∇p + μ∇²u) = 0

-∇²p + μ∇²(∇·u) = 0

비압축성 조건 적용:

∇²p = 0

속도장:

μ∇²u = ∇p
예시 해 (평판 위 흐름):
경계조건: z = 0에서 u = 0, z → ∞에서 u → (U, 0, 0)
u(z) = U(1 - e^(-√(ρU/μ)z)), v = 0, w = 0

6 수치적 접근

실제 3차원 문제 해법:
  • 유한 차분법 (FDM)
  • 유한 요소법 (FEM)
  • 스펙트럴 방법
  • Chorin's projection method (압력-속도 결합)
🖥️ 소프트웨어:
OpenFOAM, ANSYS Fluent 등을 사용한 수치해석

7 결론

  • 점성항은 μ∇²u로 추출되며, 오일러 방정식은 점성항이 없는 형태
  • 각 항을 제외한 방정식 구성 후 편미분 방정식으로 연립방정식 설정
  • 3차원 해는 해석적으로 구하기 어려우며, 특정 가정이나 수치적 방법 필요
  • 구체적인 경계 조건이나 초기 조건 필요
